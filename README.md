# SANE Deep Dives

In-depth technical explorations by SANE editors. These are detailed writeups that provide analysis and guidance but are not community-voted standards.

> **Note:** This repo is currently named `sane-av-whitepapers` for historical reasons. It will be renamed to `sane-av-deep-dives` in a future update. All URLs, navigation, and user-facing language on the site already use "Deep Dives."

## Structure

Each deep dive is a single Markdown file in this repo's root with frontmatter.
